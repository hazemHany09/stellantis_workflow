# Jeep Active Driving Assist System

**Source:** https://www.jeep.com/active-driving-assist.html  
**Date Retrieved:** 2026-04-26

## Active Driving Assist (Version 2)

Lane-Keeping Assistance feature that:
- Constantly monitors vehicle's position in the lane
- Makes subtle but constant steering adjustments
- Helps prevent vehicle from drifting to either side
- Requires driver to keep hands on wheel at all times

## Hands-Free Active Driving Assist System

Advanced version with additional features:
- Hands-Free Lane Centering Assistance
- Distance Keep Control
- Curve Speed Control
- Assisted Lane Change system
- Lane Scanning System

### Partially Autonomous Capabilities
- Hands-Free Lane Centering Assistance
- Distance Keep Control on approved roadways
- Driver monitoring via steering wheel sensors
- Driver attention monitoring camera

### Curve Speed Control
- Senses approaching curves
- Decreases vehicle speed to appropriate level
- Resumes original set speed after curve

### Lane Management
- Assisted Lane Change (with blind spot monitoring)
- Lane Scanning System (adjusts vehicle position for safety)

## Requirements

For Active Driving Assist System:
- Vehicle must be equipped with required available technology
- Jeep® Connect Vehicle Package must be enabled
- Hands-Free version requires subscription (3-year trial then paid)

## Coverage

- Over 200,000 miles of compatible roads
- Supported highways with clear road markings
- Expanding availability regularly

## Deactivation Conditions

System automatically deactivates when:
- Driver does not keep hands on steering wheel (Active Driving Assist)
- Lane markings not detected
- Brake pedal pressed
- Adaptive Cruise Control deactivated
- Turn signal used (except blind spot detection active)
- Driver performs evasive maneuver
- Driver's seat belt unbuckled
- Vehicle speed exceeds 90 mph
- Forward Collision Warning becomes active
- Driver steers out of lane
- Driver monitoring detects inattentiveness (Hands-Free version)
- Vehicle not receiving cellular signal (Hands-Free version)

## Status Indicators

Visual indicators change based on system status:
- **White:** System enabled but inactive
- **Green:** System engaged and actively steering
- **Yellow:** Driver inattentiveness detected
- **Red:** Driver inattentiveness still detected, system disengaging
- Seat vibration alerts also provided

## Adaptive Cruise Control

Works in tandem with Active Driving Assist:
- Uses radar sensors and forward-facing camera
- Maintains set distance from vehicles ahead
- Adjusts speed automatically in traffic
- Enables hands-free driving on compatible roads
