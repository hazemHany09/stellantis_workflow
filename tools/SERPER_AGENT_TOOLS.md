# Serper Tools Available To The Agent

This markdown describes the MCP tools exposed by `Serper_mcp`.

Server name: `serper-mcp`

## What The Agent Can Do

The agent can use this server for open-web search and lightweight scraping.

Typical uses:

* run general web search
* search images, videos, news
* get autocomplete suggestions



## Accessible Tools

| Tool                         | Purpose                   |
| ---------------------------- | ------------------------- |
| `google_search`              | General Google web search |
| `google_search_images`       | Google Images search      |
| `google_search_videos`       | Google Videos search      |
| `google_search_news`         | Google News search        |
| `google_search_autocomplete` | Query suggestion lookup   |

## Tool Families
